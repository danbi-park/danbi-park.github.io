# 비주얼 스튜디오 코드(Visual Studio Code)란?
비주얼 스튜디오 코드는 Microsoft가 윈도우, macOS, 리눅스용으로 개발한 **소스 코드 편집기**이다. 디버깅 지원과 Git 제어, 구문 강조 기능등이 포함되어 있으며, 사용자가 편집기의 테마와 단축키, 설정 등을 수정할 수 있다.
